# Question 1 Solution
>Input a year and find whether it is a leap year or not.

## Flow Chart

<a href="https://ibb.co/CvMW0xH"><img src="https://i.ibb.co/Xp3VJmC/Q1-Flow-Chart.png" alt="Q1-Flow-Chart" border="0"></a>

## Pseudo Code

```
start
input year
if year % 400 = 0:
    output "is Leap Year"
else if year % 100 = 0:
    output "Not Leap Year"
else if year % 4 = 0:
    output "is Leap Year"
else:
    output "Not Leap Year"
exit
```
